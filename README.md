# Tic-Tac-Toe Game
A classic Tic-Tac-Toe game developed using HTML, CSS, and JavaScript.

## Features
- Two-player gameplay (X and O)
- Automatic winner detection
- Draw detection
- Reset Game button
- New Game button
- Responsive and user-friendly interface

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## How to Play
1. Open the game in your browser.
2. Players take turns marking X and O.
3. The first player to align three symbols horizontally, vertically, or diagonally wins.
4. If all boxes are filled without a winner, the game ends in a draw.
